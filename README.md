# Vibe Coding Notes

> 用工程師的方式，讓想法快速落地。

這個 repository 用來記錄與實作我在學習 **Vibe Coding** 過程中的筆記與小專案。
目標是把 Notion 中的想法、課程內容與靈感，轉換成：
- 可執行的任務
- 可追蹤的進度
- 可版本控管的成果

---

## 📌 使用方式（給未來的自己）

- 所有想法先進 Notion
- 有「可以動手做」的內容，就在這個 repo 開一個資料夾
- 每個資料夾都回答三件事：
  1. 這個想法是什麼？
  2. 我現在能做到哪一步？
  3. 下一個最小可行行動是什麼？

---
